Kroo lets you control access at two levels: **data destinations** (your data warehouse) and **partner integrations** (Procore, Sage Intacct, Autodesk Forma, etc.).

## Admin Users

Admin users are super admins — they have unrestricted access to all destinations and all integrations automatically. No configuration needed.

## Non-Admin Users

Non-admin users have **no access by default**. Access must be explicitly granted per destination or per integration.

<Tabs>
  <Tab title="Data Destinations">
    Data destinations include warehouses like **Fabric**, **Snowflake**, and **Azure SQL**.

    | Access granted | Access not granted |
    |----------------|-------------------|
    | The user can run SQL queries against that warehouse via Kroo's **Data Viewer** | The user cannot query that destination at all |

    Each destination has a toggle in the **Permissions** tab of Settings. Grant access to let users query that specific warehouse.
  </Tab>
  <Tab title="Partner Integrations">
    Partner integrations include **Procore**, **Sage Intacct**, **Autodesk Forma**, and others.

    | Access granted | Access not granted |
    |----------------|-------------------|
    | The user can view **Pipeline Analytics** for that integration and adjust pipeline/sync settings | The integration is not visible to that user at all |

    <Tip>
    Use integration-level access to reduce mental load. For example, a controller who only works with financial data doesn't need to see Autodesk Forma pipelines.
    </Tip>
  </Tab>
</Tabs>

## How to Configure

<Steps>
<Step title="Open Permissions settings">
  Go to **Settings** > **Permissions**.
</Step>

<Step title="Find the user">
  Locate the user you want to configure in the user list.
</Step>

<Step title="Manage access">
  Under each destination or integration, click **Manage**.
</Step>

<Step title="Toggle access">
  Toggle access on or off for each destination or integration.
</Step>

<Step title="Save changes">
  Click **Save** to apply the updated permissions.
</Step>
</Steps>

# Manually Refreshing Data

If you need to manually refresh your integration data, you can do so once per day. Follow these steps:

1. Navigate to the **Home Page**.
2. Go to the **Manage Integrations** tab.
3. Click on the integration you wish to refresh.
4. Expand the accordion for the specific endpoint.
5. Click **Trigger Manual Sync**.

This will initiate a manual data refresh for the selected integration. If you encounter any issues, please contact support.
